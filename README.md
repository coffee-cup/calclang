# calclang

_HUGE WIP_

Simple programming language for a calculator. Allows creation of user defined functions along with many predefined ones.

```
// Variable def
> x = 4.0

// Function def
> x = a,b,c -> a + b + c

// Call
> fn(a,b,c)
```
